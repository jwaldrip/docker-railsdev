# Docker Rails Dev

A funcational rails environment packed with everything you need for the software
development lifecyle.

## Included Packages

* Ruby 2.1.x
* Rails Gem pre-installed
* Bundler
* Phantomjs/Casperjs for front end testing
* Node.js for fast asset compilation
